# 모두의 대시보드 (demo-dashboard-drop)

CSV 파일을 끌어다 놓으면 즉석에서 대시보드가 만들어지는 앱. **파일도 계산도 내 브라우저 안에서만** 처리된다.

- CSV 드래그 → 컬럼을 자동 분석해 **KPI 4개 + 차트 여러 개(시계열·순위 막대·구성비 도넛) + 상태 배지 표**를 한 화면에 자동 생성
- **필터 행**으로 카테고리(구분·지역·상태 등)를 좁히면 KPI·모든 차트·표가 함께 갱신
- 한글 엑셀 CSV(EUC-KR)도 알아서 인식 — 깨진 글자 없이 읽는다
- 파일이 없어도 **샘플 데이터 5종 내장**(거래 내역·카페 매출·쇼핑몰 주문·고객센터 문의·채용 지원 — 도메인 선택)으로 바로 구경 가능
- 값·집계(합계/평균/건수)를 골라 다르게 보기 + 표 미리보기. 서버 전송 0

> 이 앱은 **"회사원도 이런 앱을 만들 수 있다"** 교육용 데모 시리즈의 하나입니다.
> 원형은 실제 교육 과정의 'Drop 매출 대시보드' 실습 과제 — 수강생 결과물 한 조각이 완성 제품이 되는 과정을 보여줍니다.

## 🙋 이 앱, 나도 만들 수 있을까?

**네, 코딩을 몰라도 됩니다.** 이 앱도 AI(Claude)에게 말로 시켜서 만들었습니다.

### 따라 만들기 — 이 프롬프트를 그대로 써보세요

```
CSV 대시보드 웹앱을 만들어줘.

- CSV 파일을 드래그 앤 드롭하면 대시보드가 뜨게
- 첫 줄은 헤더, 각 컬럼 타입을 자동으로 판별(숫자·날짜·카테고리)
- 컬럼을 보고 차트를 알아서 여러 개 만들기 — 날짜가 있으면 시계열 라인, 카테고리마다 순위 막대, 구성비는 도넛
- 위쪽에 KPI 타일 4개(합계·건수·평균, 그리고 '대기/미입금' 같은 상태 컬럼이 있으면 대기 건수·금액)
- 카테고리(구분·지역·상태)를 고르는 필터 행 — 필터를 바꾸면 KPI·차트·표가 함께 갱신
- 값(어떤 숫자로 볼지)·집계(합계·평균·건수)를 골라서 바꿔볼 수 있게
- 아래에 원본 데이터 표 미리보기, 상태 값은 색 배지로
- 파일 없이 바로 볼 수 있게 샘플 데이터 5종(거래 내역·카페·쇼핑몰·고객센터·채용 등 도메인 선택) 내장
- 서버 업로드 없이 전부 브라우저에서만 처리
- 한글 엑셀 CSV(EUC-KR)도 안 깨지고 읽게
- 어두운 배경, 모바일은 세로로 쌓고 넓은 화면은 차트를 2열로
```

**포인트**: 한국 실무의 함정 — 엑셀에서 저장한 CSV는 인코딩(EUC-KR/CP949)이 달라 그냥 읽으면 한글이 깨지기 쉽습니다. 이런 함정도 AI에게 **"한글 CSV가 깨져"**라고 말하면 "인코딩을 자동으로 판별하게 해줄게"라며 해결해 줍니다.

같은 지점의 이번 달과 지난 달 매출을 나란히 막대로 비교하게.

## 📚 알아두면 좋은 개념 (비개발자용 최소 지식)

| 개념 | 한 줄 설명 |
|---|---|
| **CSV** | 엑셀이 바로 여는 표 텍스트 파일 — 시스템 간 데이터 주고받기의 공용어 |
| **문자 인코딩** | 글자를 숫자로 저장하는 규칙 — 엑셀 CSV(EUC-KR)와 웹 기본(UTF-8)이 달라 한글이 깨짐 |
| **무전송(로컬 처리)** | 파일을 서버로 안 보내고 브라우저 안에서만 계산 — 매출 데이터가 밖으로 안 나감 |
| **SVG** | 확대해도 안 깨지는 그림 형식 — 이 앱의 차트는 이미지가 아니라 SVG로 그림 |
| **KPI** | 핵심 지표(합계·평균 등) — 표 대신 큰 숫자 타일로 한눈에 보여주는 요약 |
| **배포** | 내 컴퓨터의 앱을 인터넷 주소로 여는 것. 이 시리즈는 Vercel(무료)을 씁니다 |
